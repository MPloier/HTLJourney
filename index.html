Calculator

First Project

In the first project we had to programme a calculator with simple functions such as plus, minus, times and dividing.

The calculator should also be able to clear the input.

We could think of extensions ourselves.

I extended my calculator with a key for:

  • square root
  • delete last input
  • minus sign
